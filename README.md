This repository contains everything I built while completing the **Chai aur React** series by **Hitesh Choudhary** — from React fundamentals to a full-stack production blog app with Appwrite backend, Redux state management, and React Router.

## 🛠️ Running Any Project

```bash
# Clone the repo
git clone https://github.com/prabhatyadav4/react-bootcamp.git

# Navigate to any project
cd react-bootcamp/05_bgChanger

# Install dependencies
npm install

# Start dev server
npm run dev
```

> ⚠️ For `13_ChaiBlog`, create a `.env` file with your Appwrite credentials before running.
